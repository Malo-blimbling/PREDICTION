# Ajoute Swagger/OpenAPI à ton projet Spring Boot

Ajoute la dépendance suivante dans ton pom.xml :

<dependency>
    <groupId>org.springdoc</groupId>
    <artifactId>springdoc-openapi-starter-webmvc-ui</artifactId>
    <version>2.5.0</version>
</dependency>

Après cela, tu pourras accéder à la documentation interactive sur :
http://localhost:8080/swagger-ui.html

Aucune configuration supplémentaire n'est nécessaire pour une utilisation de base avec Spring Boot 3+.
